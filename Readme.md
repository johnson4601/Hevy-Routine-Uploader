# Hevy Routine Uploader

A simple windows desktop tool for power users to update their Hevy workouts.

## Features
- **Bulk Upload**: Paste JSON workout data from Gemini (AI) directly into the app.
- **Exercise Export**: Download your full list of Hevy exercises (IDs, Muscle Groups, Equipment) to CSV.
- **Secure**: Your API key is saved locally on your machine, never sent to a third party.

## How to Use
1. **API Key**: Get your key from the Hevy App (Settings -> API).
2. **Upload**: 
   - Enter a folder name (e.g., "Winter Bulk").
   - Paste the JSON routine string.
   - Click Upload.
3. **Download**:
   - Click "Download Exercise List" to get a CSV of all available exercises.

## Installation
Just download `HevyUploader.exe` from the Releases tab and run it!
